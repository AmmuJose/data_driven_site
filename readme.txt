To install this application following files need to be copyied to the respective destination directory on the server. 
create db.inc.php(database config) file and change the path on line 16 in index.php

First make sure that following sub directories are present under /Public_html:

 /css
 /functions
 /images
 /includes
 /javascripts
 /pagelets

A sub directiory named 'uploads' should be created in the root dirctory.
